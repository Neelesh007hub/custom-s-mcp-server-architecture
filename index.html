📊 Project Overview

What We Built

A secure, cloud-based Model Context Protocol (MCP) server that enables intelligent querying of S3 data warehouses using natural language. This solution transforms complex data analysis into simple conversations.

🔍 Intelligent Discovery

Automatically scans and catalogs all data files in your S3 bucket, making them instantly queryable through natural language.

⚡ Lightning Fast

In-memory processing with DuckDB delivers sub-second query results, even for complex analytics across multiple files.

🛡️ Security

API key authentication, HTTPS encryption, and secure AWS credential management ensure your data stays protected.

🎯 Business Focused

Pre-built data models and business rules help users get meaningful insights without technical expertise.

🔧 Technical Flow

Behind the Scenes: How It Works

See the detailed technical process of how a simple user request flows through our secure system to deliver instant results.

📁 List Available Files Flow

🔍 Flow Summary:

  1. User Request: "Show me available files"
  2. Client Processing: Cursor AI sends secure HTTP request with API key
  3. Server Authentication: MCP Server verifies API key and processes request
  4. S3 Query: Server queries AWS S3 for file list
  5. Data Processing: Server creates table names and caches metadata
  6. Results Return: Formatted file list sent back to user

⚡ Query Execution Flow

⚡ Query Execution Summary:

  1. User Query: "Show me top 5 clients by revenue"
  2. Query Processing: Server converts to SQL and translates file names
  3. Database Execution: DuckDB runs query against in-memory data
  4. Results Formatting: Server formats results as clean table
  5. Display: User sees formatted results instantly
